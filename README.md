# DEFENDX Ransomware Simulation

## Educational Cybersecurity Project

This repository contains a safe, non-malicious ransomware simulation developed for the DEFENDX Cyber Security Challenge conducted by IPSR Solutions Limited.

### Features:
- Full-screen lock interface
- Password-protected unlock system (Password: defendx123)
- Attempt counter for wrong passwords
- Educational warning messages
- No real encryption or file damage

### Technologies Used:
- Python 3
- Tkinter GUI Library
- Kali Linux 2025.2
- Windows 11

### Ethical Notice:
This project is for EDUCATIONAL PURPOSES ONLY. It does not perform any real encryption, file modification, or system damage. Use only in controlled lab environments.
